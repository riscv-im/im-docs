# Welcome

## 简介

当下，微信、Telegram等即时通讯系统已经成为了个人或者组织在互联网上进行信息交换的重要工具。但这些实时通讯系统或多或少都存在着一些问题，比如：信息同步不及时、不支持已读未读功能、提及跳转操作不便、UI界面不够简洁等。本项目采用django和react作为框架，旨在为用户提供基于web的，更加简洁、高效、便捷、安全的即时通讯系统。

清华大学2023年秋季学期软件工程项目。
